# 实习心得

开始这周开始开发公司的BI数据后台项目，项目经理开会跟我解释UI设计图的功能细节以及布局细节。会后，我的主管耐心地对我说，现在写的项目与在学校中写的Dome不一样，需求会更多，要求会更严格。无意中我也感受到了些许压力，毕竟之前在学习写的项目是略为粗糙的项目，比公司需求的项目要求要少一些，着重点在与是否可以用技术栈实现项目。而现在不仅要用技术栈实现项目，而且还要将技术栈用的更熟练，将项目的需求做的更好。让我更有压力的是，这个项目是由我从头开始搭建的，也就是说这个项目是我第一个企业项目。虽然说对于第一次做公司项目的我来说心里有些忐忑不安，怕自己的代码是一座“屎山”，更怕自己没办法实现项目的需求，但是对于自己的职业生涯来说无疑是一次收获颇多的锻炼机会。
