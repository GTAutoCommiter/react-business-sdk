# 实习心得

本周完成了公司BI数据中控前端项目的五个页面，基本上一天能完成一个Bi前端页面的布局以及页面中的所有功能。相比于上周的自己，明显感觉到自己的React技术有所提高，自己产出的效率更高了。我相信这或许与我每天利用通勤时间，在地铁上拿着kindle电子书看前端相关的专业书籍有关。虽然书中的所有知识不能完全记得一清二楚，起码大概知道有哪些技术栈可以让我更好的完成公司的业务。在工作时，尤其感受到这一点，例如AntdProUI组件库，先大致过一遍所有的组件，大概知道有哪些组件能实现哪些功能，然后当我对照UI设计需求时，就能更好更快的找到对应的组件实现相关的功能。以此类推，计算机专业的所有技术都是如此，专业书籍特别多，不可能每本书都精读一遍，但是可以将一本书大致过一遍了解某个技术栈大概能干些什么能实现些什么，然后再根据相关的需求仔细去研读其具体的用法。
