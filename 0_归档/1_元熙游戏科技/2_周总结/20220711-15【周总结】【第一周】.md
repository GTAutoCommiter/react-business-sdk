#### 1. gitLab提交流程不清楚

1. 先将仓库clone 在本地
    
     git clone '仓库地址'
    
2. 将仓库中的代码更新
    
     git pull
    
3. 将改完的代码提交到缓存区
    
     git add .
    
4. 将缓存区中的代码写上提交信息
    
     git -m ''
    
5. 将缓存区的代码提交到远程仓库
    
     git push
